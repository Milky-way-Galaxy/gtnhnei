# GTNH NEI

A lightweight NEI-style recipe and usage searcher for GregTech: New Horizons.

Features:
- Search GTNH items and fluids
- Left click / tap: recipe
- Right click / double tap: usage
- Hold: copy exact name
- Real shaped recipe slot positions
- Machine recipe tabs
- Back/forward navigation
- Mobile-friendly UI

Uses parser/data structure from ShadowTheAge/gtnh:
https://github.com/ShadowTheAge/gtnh
